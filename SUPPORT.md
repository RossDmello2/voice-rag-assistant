# Support

Use GitHub issues for public support questions, bug reports, and feature requests:

- Bug reports: use the `Bug report` issue form.
- Feature requests: use the `Feature request` issue form.
- Questions: open a normal issue and label it `question` when available.

Do not include private documents, `.env` values, API keys, tokens, local SQLite data, model artifacts, or screenshots containing credentials in public issues.

For vulnerabilities, use GitHub private vulnerability reporting:

https://github.com/RossDmello2/voice-rag-agent/security/advisories/new
