# Project Assets

This folder contains public documentation assets for VoiceRAG Agent.

## Files

- `screenshots/home.png`: README hero screenshot of the running UI.
- `screenshots/auth-documents.png`: screenshot focused on auth and document-management flows.
- `screenshots/main-workflow.png`: sanitized chat workflow screenshot captured with local route interception.
- `screenshots/mobile.png`: mobile viewport screenshot for responsive-layout evidence.
- `social-preview.png`: 1280x640 repository preview image for GitHub sharing.

Regenerate screenshots after significant frontend visual changes. Do not place secrets, `.env` values, real private documents, local database content, or provider keys in screenshots or demo media.
